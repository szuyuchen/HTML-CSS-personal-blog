如何製作 GitHub Profile README ?


2022/10/11 README


如果想要把 GitHub 個人頁面設計的吸引人該怎麼做呢？

其實不難, 加個「README.md」就可以了！


製作方法：


1) 新增 Repository

github 個人主頁面點選上方列的「Repositories」 -> 點擊右上藍色「NEW」按紐。


2) 建立與自己 github 帳號同名的 Repository

Repository name 欄位中輸入與自己 github 帳號 (owner) 同名稱 -> 點選「Public」 -> 勾選「Add a README file」-> 點選「Create repository」。


3) 客制 README.md 內容

在 Repository name 欄位中輸入與自己 github 帳號 (owner) 同名稱 -> 點選「Public」-> 勾選「Add a README file」-> 點選「Create repository」。

Edit file 旁邊有個眼睛符號的「Preview」可確認預設狀態。


4) 新增 README.md 內容

下方的 Commit changes 欄中輸入「新增改動的內容」-> 點擊「Commit changes」即可完成。
