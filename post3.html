10 種 VSCode 外掛擴充套件 (Extensions) 介紹


2022/10/11 VSCode


這篇提供 10 種 VSCode 外掛擴充套件 (Extensions) 相關介紹。

安裝方式:


1. 下載 VSCode 編輯器：

微軟開發的一款免費程式碼編輯器 (Visual Studio Code), 有很多擴展套件可搭配做使用。
前往 [Visual Studio Code]下載適合的版本。


2. 安裝外掛擴充套件:

安裝方法：點擊 VSCode 版面左側的「Extensions」-> 彈出的欄位中輸入套件名稱 -> 點選套件名稱(選置頂的套件被安裝次數最高) -> 點擊「install」即可。


1) Auto Rename Tag

寫程式碼時變更標籤時,結尾標籤能自動修改,非常省時。


2) JavaScript(ES6) coding snippets

可以縮寫快速打出指令。
使用：輸入「clg」按「->|」(Tab鍵) 即可跑出 console.log(object)。


3) Live Server

在本地開發的檔案儲存的同時,可在瀏覽器裡同步看到本地開發的程式碼,達到即時預覽效果。
使用：點擊 VSCode 版面底部右下方的「Go Live」。


4) Polacode

快速製作程式碼圖,可簡單將程式碼轉存成圖片。
使用：安裝此套件後,點選需要的程式碼區域,輸入「command shift p」,即可打開搜尋列,輸入「Polacode」再按下「enter」即可生成程式碼圖。


5) Prettier - Code formatter

自動排版,省時不用手動對齊。


6) Live Share

如果出現在編輯器上需要共同處理同一個程式碼檔案的狀況，可將自己編輯器的控制權分享給團隊成員，也可以同時編輯此程式碼檔案。


7) Quokka.js

實時觀看 javascript 的變數的變化。


8) GitLens

在程式碼後方可以看到 commit 詳細資訊,顯示何時提交或更改,和更改者的資訊。


9) Git History

方便查看 Git 修改紀錄和檔案歷史。


10) ESlint

程式碼的不良風格的地方會顯示紅線,能為程式碼增加規範。
